# SamplifyPlus JUCE Migration Plan
# Migrating from old JUCE (5.x/6.x) to modern JUCE 7+

## Overview
The project was originally developed for Windows/macOS with older JUCE versions.
Goal: Make it compile on Linux with modern JUCE 7+.

## Phase 1: Project Configuration [COMPLETED]
1. Add Linux Makefile exporter to SamplifyPlus.jucer
2. Set correct module paths for Linux (/home/jakee/Documents/juce/modules)
3. Disable useGlobalPath for all modules

## Phase 2: Fix Deprecated/Removed APIs [COMPLETED]
Major API changes in JUCE 7+:
1. Modal operations removed on Linux - must use async patterns
2. FileChooser::browseForDirectory() -> launchAsync()
3. CallOutBox::runModalLoop() -> launchAsynchronously()
4. DialogWindow::showModalDialog() -> enterModalState()
5. PopupMenu::show() -> showMenuAsync()
6. NativeMessageBox::showYesNoBox() -> showAsync() with MessageBoxOptions
7. Window::runModalLoop() -> enterModalState() with callback

## Phase 3: Install Dependencies [PENDING]
Required packages for Linux build:
- libfreetype6-dev (FreeType font rendering)
- libwebkit2gtk-4.1-dev (WebKit for web views)
- libgtk-3-dev (GTK3 for UI)
- libasound2-dev (ALSA for audio)
- libcurl4-openssl-dev (already installed)

## Phase 4: Build and Test [PENDING]
