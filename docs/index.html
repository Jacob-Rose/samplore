Samplore

A new way to explore your personal sample folders

Now Open Source! Built with JUCE for Windows, macOS, and Linux

Learn More View on GitHub

What Is It?

You know that new sound you're looking for? Well listen to this.

Samplore is a complementary program that works with the DAW you already use every day to make music. Samplore features near-latency-free sample previewing from any point in the sample. Try cutting off the start of a sample and see if it could work, try listening to it backward. Samplore is made to be a playground to inspire, then once inspired an easy way to transfer your new sound into your DAW.

Samplore allows you to organize your samples over time and help save sounds for later if it's not fitting the current song, but you don't want to lose it.

Features

Open source sample library management for music producers

Who Is It For?

Music producers with folders full of unexplored sample files. A better way to search for samples in these folders, and then help organize these samples and leave notes for later use.

Easy and Simple to Use

Made to replicate the feeling of going through a record store finding sounds. Search through sample packs downloaded online, or even the sample folder you have built up as your time as a producer.

Works with Existing Music Library

Samplore is free and open source. It works with your pre-existing sample library directory or multiple directories. Built with JUCE for cross-platform compatibility.

Ignite Your Creativity

Samplore's goal is to be the middleman between your dedicated DAW and your sample library.

Quick Search

Search for keywords and tags faster than Windows Explorer with better results

Directory Selector

Directory Explorer can be used to set which files are in the active sample pool

No-Loadtime Preview

Samples start playing almost instantaneously after clicked - no more waiting for loading!

Tags

Bookmark sounds with user-defined tags. Organize your sounds over time as you search.

Drag and Drop

Found a sound that inspires you?
Simply drag it from the application into your DAW or any other external program.

Randomize Search

Challenge yourself or just find something new!
